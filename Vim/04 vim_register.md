## 复制与粘贴

操作：
- `p` 粘贴到光标之后；`P` 粘贴到光标之前
- `xp` 调换光标之后的两个字符
- `ddp` 调换当前行和他的下一行


**寄存器**：`{register}` 前缀指定，若不指明，缺省使用无名寄存器
```
"ayiw 当前单词复制至寄存器 a
"bp 粘贴自寄存器 b
```

寄存器类型：
- `"_` 黑洞寄存器，有去无回
- `"0` 复制专用寄存器
- `"p"` 系统寄存器

## 寄存器替换、插入
- 在可视模式下使用 `p` 命令， Vim 将使用指定的寄存器内容替换高亮选区中的文本

- 在插入模式下：
	- `<C-r>0` 或 `<C-r>"` 插入寄存器内容，替换 `puP` 和 `Pup` 的
